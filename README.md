# ヤジュッター検索

ヤジュッター (https://yajutter.yajuvideo.in/) のヘッダーに検索機能を追加する Firefox 拡張機能です。

[有志の方が作ってくださったChrome版](https://github.com/kongyo2/yajutter-kensaku-chrome)


## 機能

- ヘッダーに検索バーを追加
- 投稿・ユーザーをリアルタイム検索
- 検索結果をドロップダウンで表示（ユーザーアイコン・名前・いいね数付き）
- 日本語入力（IME）対応
- 検索結果のキャッシュによる高速化

## インストール

1. [Releases](../../releases) ページから最新の `.xpi` ファイルをダウンロード
2. Firefox で `about:addons` を開く
3. 歯車アイコン → 「ファイルからアドオンをインストール」→ ダウンロードした `.xpi` を選択

## 使い方

### 1. APIキーの設定

1. Firefox で ヤジュッター を開く
2. ヤジュッターAPIを入力する画面があるので自分のAPIを入力し、保存をクリック
3. ヤジュッター検索の設定からもAPIキーを入力できます。

### 2. 検索

1. [ヤジュッター](https://yajutter.yajuvideo.in/)を開く（設定後はページをリロード）
2. ヘッダーに表示された検索バーにキーワードを入力
3. 検索結果がドロップダウンで表示されるので、クリックして投稿に移動
4. バグったときは大体リロードすれば直ります。

## 動作環境

- Firefox 140 以降
